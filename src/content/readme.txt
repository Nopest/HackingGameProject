Le dossier "contenu" comporte toutes les données du jeu que l'aglorithme de génération d'une partie utilisera.
Il comprend donc :
	- Les noms de machines : PC, serveur, proxy
	- Les IP : fichier contenant des adresses IP à attribuer
	- Les fichiers : fichiers de base pour un pc, serveur, proxy
			 fichiers spéciaux (easter eggs, logs irc, .txt, .exe, ect...)
	- mot de passe : fichiers de différents mot de passe
	- login : fichier de différents login
